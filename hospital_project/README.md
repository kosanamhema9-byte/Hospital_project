# Hospital Management Project (Backend + Frontend)

Folders:
- hospital-backend: Node.js + Express + MongoDB backend
- hospital-frontend: React frontend (Vite)

## Quick start
1. Start backend
   - cd hospital-backend
   - copy .env.example to .env and fill MONGO_URI
   - npm install
   - npm run dev
2. Start frontend
   - cd hospital-frontend
   - npm install
   - npm run dev

Make sure MongoDB is accessible (Atlas or local). Backend runs on :5000, frontend on :3000.
